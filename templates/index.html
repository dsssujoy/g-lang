G-Lang Documentation

            
Welcome to G-Lang!

            
G-Lang is a quirky, Banglish-inspired programming language built with Python. It is designed for
                beginners
                and coding enthusiasts who want to learn programming with fun. It supports variables, loops,
                conditionals,
                functions, input/output, and playful syntax with funny error messages.

        


        

            
1. Getting Started

            
All G-Lang programs start with shuru and end with ses.

            

                shuru 

                  bol "Hello, world!" 

                ses
            

        


        

            
2. Variables

            
Declare variables with dhoro and assign values using =.

            

                dhoro x = 10 

                dhoro name = "Sujoy" 

                x = x + 5
            

            
Supports numbers, strings, and boolean values (thik for true, dhop for false).
            

        


        

            
3. Print Statements

            
Use bol to print values or messages.

            

                bol "Hello G-Lang" 

                bol " is my number"
            

        




        

            
4. Conditional Statements

            
Use jodi for if, naile for else.

            

                dhoro age = 20 

                jodi age > 18 { 

                  bol "Adult" 

                } 

                naile { 

                  bol "Child" 

                }
            

        


        

            
5. Loops

            
While loop uses jotokhon, for loop uses ghurche.

            
While Loop:

                dhoro i = 0 

                jotokhon i < 5 { 

                      bol i 

                      i = i + 1 

                    }
            

            
For Loop:

                ghurche dhoro i = 0; i < 3; i=i + 1 { 

                      bol i 

                    }
            

        


        

            
6. Functions

            
Define functions with kaj and return values with phire.

            

                kaj add(a, b) { 

                  phire a + b 

                } 


                dhoro result = add(5, 10) 

                bol result
            

        


        

            
7. Boolean Values

            
Use thik for true and dhop for false.

            

                dhoro flag = thik 

                jodi flag { 

                  bol "Yes!" 

                }
            

        


        

            
8. Blocks

            
Use shuru ... ses to group statements into a block.

            

                shuru 

                  dhoro x = 5 

                  bol x 

                ses
            

        


        

            
9. Operators

            
G-Lang supports:

            

                Arithmetic: +, -, *, / (integer division)

                Comparison: <, >, ==

                String concatenation: +
            

        


        

            
10. Supported Features

            

                - Variable declaration and assignment

                - Arithmetic and string operations

                - Conditional statements (if-else)

                - Loops (while, for)

                - Functions with parameters and return values

                - Boolean values (thik, dhop)

                - Blocks (shuru ... ses)
            

        


        

            
11. Limitations

            

                - Floating point numbers not supported (only integer division)

                - No arrays, lists, or dictionaries

                - No classes or OOP

                - No advanced operators (+=, -=, %, etc.)

                - No file operations or imports
            

        


        

            
12. Full Example Program

            

                shuru 

                  bol "Welcome to G-Lang!" 

                  dhoro name = "Sujoy" 

                  bol "Hello " + name 

                  dhoro i = 0 

                  jotokhon i < 3 { 

                        bol "Loop iteration: " + i 

                        i = i + 1 

                      } 

                      kaj add(a,b) { 

                        phire a + b 

                      } 

                      dhoro result = add(10,5) 

                      bol "10 + 5 = " + result 

                    ses
